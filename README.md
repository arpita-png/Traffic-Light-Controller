# Traffic Light Controller

Embedded traffic light controller using TM4C123GXL microcontroller.

## Features
- Timer-based traffic sequencing
- Pedestrian button support
- State-machine logic
- GPIO control

## Demo Video
(https://www.youtube.com/shorts/sg4RAKcW4iE)

## Technologies Used
- Embedded C
- TM4C123GXL
- Code Composer Studio

## Project Structure
- main.c → main application logic
- traffic_light.c → traffic light control logic
- traffic_light.h → header definitions
